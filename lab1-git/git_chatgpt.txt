#### R1-R2：初始化仓库、添加并提交

#### R3-R5：修改文件并提交
```bash

```

#### R6：撤销上一次提交
```bash

```

#### R7：查看提交记录

### 四、分支管理（实验场景2）

#### 操作示例：

```bash
git branch           # 查看分支
git checkout -b B1   # 创建分支B1
git checkout -b B2
git checkout -b C4 B2

# C4 分支提交
git add .
git commit -m "C4分支修改"

# B1 分支冲突合并
git checkout B1
git merge C4         # 若有冲突，手动解决
git add .
git commit -m "合并C4到B1，解决冲突"

# 删除合并分支
git branch -d C4

# 创建学号分支
git checkout -b 20251234
git merge B2
git push origin 20251234
```
